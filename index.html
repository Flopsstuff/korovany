What is this? ​

Korovany is a 3D action game that runs entirely in the browser as a single-page application. This site is the project's living documentation — kept in the repo under docs/ and published automatically.

  • Live app: https://korovany.aimost.pl/
  • Source: https://github.com/Flopsstuff/korovany
  • Start with the Getting started guide, then read the Project rules.
  • Future level direction lives in the World specs.
